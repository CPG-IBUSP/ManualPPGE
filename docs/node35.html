Curso preparatório para ingressantes

É curso preparatório para o exame de ingresso no mestrado e doutorado do Programa de Pós-graduação em Ecologia, destinado para pessoas contempladas pelas políticas de ações afirmativas do programa. É mais uma iniciativa dos estudantes do PPGE-USP, para qual todos os estudantes podem contribuir.
